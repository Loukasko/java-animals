PROJECT TITLE: School Project JAVA 
PURPOSE OF PROJECT: Java basics 
VERSION or DATE: 4/2015
AUTHORS: Loukas Kormazoglou
COMMENTS: Basic implementation of object-oriented programming . The project contains basic usage of swing class for graphics. 
USE CASE : The user starts the program and the dimensions of a custom grid are requested. The grid contains 4 kind of objects (grass,empty,animal_type_1,animal_type_2). During each round animals can move up to 2 blocks randomly and according to where they land , they interact with each other(empty->nothing,animal_type_1->(eats)grass , animal_type_2->(eats)animal_type_1) . In each round stats of the progression are displayed.
